SINTAXIS BASICA DEL LENGUAJE
Expresion: Puede ser un átomo o lista

Átomo : Puede ser un numero o string o identificador o carácter o boolean
Elemento basico -> Identificador, letras y dígitos

Lista: ‘(‘secuencia-expresion’)’
(Conjunto de elementos entre paréntesis): formado por atomos y listas

Secuencia-expresion : expresión secuencia-expresion | expresión

Función:  (nombre_funcion argumento1, argumento2, argumenton)

COMPARADORES ARITMETICOS DIFERENTES A OTROS LENGUAJES
Even? ¿Es numero par?

odd? ¿Es numero impas?

Zero? ¿Es cero el numero?

Ejemplo: (zero? 1) -> false

FUNCIONES PARA MANIPULAR LISTAS

Car: Devuelve el primer elemento de la lista 
(car ‘(A B C)) -> ‘A

Cdr: Devuelve la lista eliminando el primer elemento
(cdr’(A B C)) -> ‘(B C)
